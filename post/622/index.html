8大基本数据类型

  1. byte
  2. short
  3. int
  4. long
  5. float
  6. double
  7. char
  8. boolean

计算机的存储单位是什么

  1. 计算机只认识0和1
  2. 计算机的存储设备有两种：内存、磁盘（文件）
  3. 代码本身以文件的形式存储在磁盘（永久存储）
  4. 代码中定义的变量，运行时存储在内存（临时存储）
  5. 计算机的最小存储单位是比特（bit），称为一个比特位或者1个二进制位
  6. 计算机的最小内存申请单位是字节（byte），1个字节=8个比特
  7. 单位换算
    1byte =8 bit
    1kb=1024 byte
    1mb= 1024kb
    1gb,1tb,1pb.....
    kb千字节 mb兆字节 gb 吉字节 tb太子节 pb 拍字节

计算机如何存储整数

byte类型：1个字节（8位）存储整数
short类型：2个字节（16位）存储整数
int类型：4个字节（32位）存储整数
long类型：8字节（64位）存储整数

最高位是符号位：0正1负
除符号位外，都是数据位

计算机如何存储小数

计算机通过在二进制位中加一个浮点来表示小数，
单精度浮点类型float 需要4字节 精确到6位
双精度浮点类型double 需要8字节，精确到15位

超过就丢失精度

计算机如何存储布尔

布尔类型（boolean）表示真和假（ture or false），来做条件判断。
true用0存储，false用1存储。（代码中）

计算机如何存储字符

  1. 计算机不认识任何符号
  2. 全靠各种各样的编码表来认字，所以要记忆一些表。（被人为规定好的）
  3. 存储字符用char类型

字符和字节的关系

  • 字符编号在128内，只需要1个字节
  • 字符编号在128以上存储时需要2+个字节

如何存储字符串

  1. 字符串是一个字符的序列或者集合
  2. char只能存储单字符，字符串用String类型存储
  3. String s = "Hello";实际上是存取一个序列[ 'H','e','l','l','o']
  4. char用单引号存储，string用双引号存储
  5. char属于8大基本类型之一，String不属于基本类型

计算机如何存储文件、图片、音频、视频

综上所述都是以二进制存储的
eg.

  1. 图片中的一个像素就是一个数字
  2. 音频中的一个音波就是一个数字
  3. 视频=图片+音频

逻辑运算符

与或非
本来时&& 与，II或，非！
&是不短路，I也是不短路，就是真假如何都还要把逻辑运算，判断完
&&就是如果第一组已经出了结果就不用管后面的运算了，所以后面的就不算了
经典案例：
result =x++>10 & --y < 20
result = x++>10 && --y < 20
参照 Demo06

位运算

  1. 按位与（5 &9)
    把两个数转化为二进制，对齐位数，都为1，那一位才为1。
  2. 按位或 (5 I 9)
    同上有1个是1，那1位就算1
  3. 按位异或(5^9)
    两数对齐，相同为0，相异为1
  4. 按位非(~ 5)
    按位取反，包括符号位

注意计算的时候注意符号
计算机存储的是二进制的补码
换算十进制用原码
正数：原码 = 反码 =补码
负数：原码 取反 = 反码+1 =补码

原码取反：除了符号位按位取反
按位非：包括符号位在内都要取反。（算出来的是补码）
