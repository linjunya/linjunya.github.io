程序执行流程

1. 顺序结构

默认的执行结构 从上往下（主函数）

2. 分支结构

  • 单if分支

condition代表条件

  1. boolean类型的变量，eg.Boolean flag = true，flag变量放在condition处

  2. 运算结果是Boolean的表达式，x>100

  3. 返回值结果式boolean类型的函数调用

  4. 直接给予true或者false（太直白了）

    if (condition) {

    }
    //判断：分支以内的就是分支结构的代码，以外的都是顺序结构

if-else分支

if (condition) {

} else {

}
//这个要注意的是条件为真就执行if后面的代码，为假就把else后面的代码执行了。

多路if分支

// 顺序结构中的代码
if (condition1) {
// 分支结构中的代码 （condition1为真执行）
} else if (condition2) {
// 分支结构中的代码 （condition2为真执行）
} else if (condition3) {
// 分支结构中的代码 （condition3为真执行）
} else {
// 分支结构中的代码 （以上condition都不为真执行）
}
// 顺序结构中的代码

//多路if分支要注意的就是虽然条件写这么多，实际上是在筛选，把苛刻的条件都放在前面，看起来很多但只会执行一条分支上的代码。

嵌套分支

// 顺序结构中的代码
if (condition1) {
// 分支结构中的代码 (condition1为真执行)
if (condition2) {
// 分支结构中的代码 (condition1为真&&condition2为真执行)
} else {
// 分支结构中的代码 (condition1为真&&condition2为假执行)
} 
} else {
// 分支结构中的代码 (condition1为假执行)
}
// 顺序结构中的代码

//顾名思义，嵌套，大的分支里面还可以塞一个分支，外面的循环一次，里面的循环一轮

switch-case分支（只适合等值判断，只执行一个case，且一定执行一个case，case后break必须有）变量类型 支持整型除了long，..字符，字符串，枚举。float，double，Boolean都不行

switch (variable) {
case 值1:
// 分支1 （ variable == 值1  ）   if ( variable == 值1)
break;
case 值2:
// 分支2 （ variable == 值2  ）   else if ( variable == 值2)     
break;
case 值3:
// 分支3 （ variable == 值3  ）   else if ( variable == 值3)
break;
default:                             
// 分支4 （ variable 不等于上面任意case中的值  ） else 
break;    
}

//switch主要用于等值判断，switch括号里跟着一个变量，case 1，2，3什么乱七八糟的都是跟这个变量有关的条件这个变量等于case后面的值，然后发生的故事，执行的命令。一样的也跟那个多支if一个意思在筛选的感觉。只会执行一个分支上的代码。但是switch的判断主要是**等值！**

3. 循环结构

  • 循环四要素
  1. 循环的起点（变量，不参与循环）
  2. 循环的终点（条件，满足了就偃旗息鼓）
  3. 循环体（位于循环结构里的代码）
  4. 循环的迭代（每循环一次，循环变量的变化，这里可以是自增自减也可以是一个描述变化的表达式）

注意：如果循环未完成迭代，一直没变化就会造成死循环。

while 循环

int x = 0; // 1.定义变量x作为循环的起点
while (x < 10) { // 2.x < 10 是循环的终点,他是一个condition条件
System.out.println(x); // 3.循环体
x ++; // 4.循环的迭代
}

//感觉跟if很像，括号跟条件，然后一个大括号就放代码（你喜欢的话，就叫循环体也行方便你融入循环这个概念）

// 1-100
public class Demo07 {
public static void main(String[] args) {
// 练习：写一个循环，计算1-100之间的偶数之和，并打印
int sum = 0; // 用于在循环过程中累加n
int n = 0;
while (n <= 100) {
sum += n; // sum = sum + n;
n += 2; // n = n + 2;
}
System.out.println ("sum:" + sum);
}
}

for 循环

for (变量;条件;循环迭代) {
循环体
}
// 用for计算1-100偶数之和
int sum = 0;
for (int n = 0; n <= 100; n += 2) {
sum += n;
}
System.out.println ("sum:" + sum);

do-while循环

变量
do {
循环体
循环的迭代
} while (条件);

// 用do-while计算1-100偶数之和
int sum = 0;
int n = 0;
do {
sum += n;
n += 2;
} while (n <= 100);

break和continue

  • break用于跳出循环
  • break用在switch-case结构中用于跳出switch
  • break用在循环结构中用于跳出循环 (即在循环终点之前，提前结束循环)
  • continue用于结束本次循环 （本次循环后面的代码不执行），继续下次循

break是直接跳出循环，continue是跳过这一次循环，下一次循环还是要继续循环下去的要把没做完的事儿做完，相比较起来，break更像是撂挑子就不干了，continue像是先递上辞职信，然后再交接工作的感觉。

就近原则 (如果有多层循环，作用于最近的一个循环身上)

for () {
for () {
if () {
// break在内层循环中, 用于跳出内层循环, 不影响外层循环
break;
}
}
if () {
// break在外层循环中, 用于跳出外层循环
break;
}
}

函数结构

函数是什么

  1. 函数的英文单词function，也称为方法method
  2. 函数是程序的最小执行单元
  3. 类是程序的最小文件单元，一个类中可以包含多个函数
  4. 一个函数可以看作是一个可被复用的过程或者一个任务
  5. 函数需要先定义后调用，一次定义，到处调用

函数的五要素

  1. 函数名

  2. 函数的主体

  3. 函数的参数

  4. 函数的返回类型

  5. 函数的返回值

    函数的返回类型④ 函数名① (参数列表③) {
    函数体②
    函数的返回值⑤
    }

定义函数时需要注意的地方

一个函数不可以定义在另一个函数的 {} 里面，函数与函数必须是平级关系。
