Java新手注意

Java的类名命名规则

Java 中定义类名的规则很宽松。名字必须以字母开头，后面可以跟字母和数字的任意组合。 长度基本上没有限制。但是不能使用 Java 保留字（例如，public 或 class）作为类名。

驼峰命名法

如果名字由多个单词组成，每个单词的第一个字母都应该大写（这种在一个单词中间使用大写字母的方式称为大驼峰法 -- 即帕斯卡命名法。eg.World、Hello.
