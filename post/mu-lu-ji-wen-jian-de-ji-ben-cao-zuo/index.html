目录及文件的基本操作

2024-04-04

一、pwd命令

  • （Print Working Directory）
  • 描述：显示出完整的当前工作目录的名称 绝对路径
[root@localhost ~]# pwd			#显示当前的工作目录
/root
  • 选项：-P 显示链接的真实路径
[root@localhost ~]# cd /bin/	#切换到 根目录下的 bin 目录
[root@localhost bin]# pwd
/bin
[root@localhost bin]# pwd -P	#显示链接的真实路径
/usr/bin

二、cd 命令

  • （Change Directory）
  • 描述：切换当前的工作目录
  • 命令格式：cd [目标文件夹位置]
[root@localhost ~]# cd /opt/
[root@localhost opt]# pwd
/opt
cd 命令常用的几种方式
  1. 直接使用 cd 会切换到当前登录用户的家目录、
[root@localhost opt]# cd 
[root@localhost ~]# pwd
/root
  1. cd .. 切换到当前目录的父目录
[root@localhost ~]# cd ..
[root@localhost /]# pwd
/
  1. cd ~用户名 切换到目标用户的家目录
[root@localhost ~]# useradd lisi
[root@localhost ~]# cd ~lisi
[root@localhost lisi]# pwd
/home/nsd2005
  1. cd - 切换到上一个工作目录
[root@localhost ~]# cd /opt 
[root@localhost opt]# cd /etc
[root@localhost etc]# cd -		#频繁的在两个目录中切换时 很好用
/opt
  • 绝对路径和相对路径
    • 绝对路径：以根目录开始的完整路径
    • 相对路径：以当前所在的目录为参照物的路径

三、ls 命令

  • （list）显示指定目录下的内容（列出当前工作目录所含文件及子目录)

  • 命令格式：ls [选项] [目录或文件名]…

[root@localhost ~]# ls
anaconda-ks.cfg       公共  视频  文档  音乐
initial-setup-ks.cfg  模板  图片  下载  桌面
常用的选项
  • -A 显示目标目录下所有的文件和目录，包括隐藏文件
[root@localhost ~]# ls -A /root/
#除 -A 外 还有个 选项是 -a
#两个选项的区别在于   -a  会多显示   . 和  ..
#. 和  ..  是两个特殊的目录，分别代表 当前目录和上一级目录
  • -l 长格式显示文件的详细属性
[root@localhost ~]# ls -l /etc/passwd
-rw-r--r--. 1 root root 2727 6月   8 10:24 /etc/passwd
  • -d 显示目录本身的属性
[root@localhost ~]# ls -ld /etc
  • -h 提供一个易读的单位
[root@localhost ~]# ls -lh /etc/passwd
  • -R 递归显示内容
[root@localhost ~]# ls -R /etc/
  • -u 显示文件或目录最后被访问的时间
[root@localhost ~]# ls -l /etc/passwd
[root@localhost ~]# ls -lu /etc/passwd
  • -t 以修改时间排序，ls目录默认是按文件名称排序的
[root@localhost ~]# ls -lt /etc/

四、mkdir命令

  • 创建目录

  • 命令格式：mkdir [选项]... 目录...

[root@localhost ~]# mkdir /opt/nsd2005
[root@localhost ~]# cd /opt/nsd2005
[root@localhost nsd2005]# pwd
/opt/nsd2005
常用的选项
  • -p 创建一个目录并将它的父目录也创建出来（创建一个连续的目录）
[root@localhost nsd2005]# mkdir -p /opt/nsd2005/aa/bb/cc/dd/ee
[root@localhost nsd2005]# cd /opt/nsd2005/aa/bb/cc/dd/ee
[root@localhost ee]# pwd
/opt/nsd2005/aa/bb/cc/dd/ee
  • -m 创建目录时指定权限
[root@localhost ~]# mkdir -m 444 /opt/nsd05
[root@localhost ~]# ls -ld /opt/nsd05
dr--r--r--. 2 root root 6 6月   8 10:50 /opt/nsd05

五、touch命令

  • 创建文件或修改文件时间
    • 不存在则创建，存在则跟新文件所有的时间为系统时间
  • 命令格式： touch 文件名
[root@localhost ~]# touch 1.txt   		#在当前目录下创建文件1.txt
[root@localhost ~]# touch /opt/1.txt    #在指定目录下创建文件1.txt

六、cp命令

  • （copy）
  • 描述：复制文件或目录
  • 命令格式：cp [选项]... 原文件1、原文件2… 目标路径
[root@localhost ~]# cp /etc/passwd /opt/			#拷贝/etc/passwd到/opt下
[root@localhost ~]# ls /opt/
 
[root@localhost ~]# cp /etc/passwd /opt/pass		#拷贝/etc/passwd到/opt下并将其副本改名为pass
[root@localhost ~]# ls /opt/
常用选项
  • -r 递归复制，复制目录时必须有的选项
[root@svr7 ~]# cp /home/ /opt/						#复制目录时，必须要加-r选项
cp: 略过目录"/home/"
[root@svr7 ~]# cp -r /home/ /opt/
  • -p 保持权限不变的复制
[root@svr7 ~]# cd /var/named/				#切换到地址库文件存放的目录
[root@svr7 named]# ls -l named.localhost
[root@svr7 named]# cp  -p   named.localhost      tedu.cn.zone   
#复制模板文件，-p选项保证文件属性（所属组为named）不变
[root@svr7 named]# ls -l tedu.cn.zone
  • -a 复制时保留源文件的所有属性（包括权限、时间等）
[root@svr7 ~]# ls -l /etc/passwd
-rw-r--r--. 1 root root 2237 9月  17 19:51 /etc/passwd
[root@svr7 ~]# cp /etc/passwd /opt/
[root@svr7 ~]# ls -l /opt/passwd 			#直接复制的结果
-rw-r--r--. 1 root root 2237 9月  18 22:19 /opt/passwd
[root@svr7 ~]# cp -a /etc/passwd /opt/pass
[root@svr7 ~]# ll /opt/pass					#使用-a选项复制的结果
-rw-r--r--. 1 root root 2237 9月  17 19:51 /opt/pass

七、rm命令

  • 删除目录或文件

  • 命令格式：rm [选项]... 文件...

  • 常用的选项

    • -r ：递归删除（删除目录必须要有）
    • -f ： 强制删除
    [root@localhost ~]# rm -rf /opt/1.txt 
[root@localhost ~]# rm -rf /opt/nsd05/
[root@localhost ~]# ls /opt/

八、mv命令

  • （move）移动文件或目录

  • 路径不变的移动（重命名）

  • 命令格式：mv 原文件… 目标路径

    [root@localhost ~]# touch 1.txt
[root@localhost ~]# mv 1.txt  /opt/1.txt

  • 重命名

    [root@localhost ~]# mv  /opt/1.txt /opt/2.txt
[root@localhost ~]# ls /opt/

九、du 命令

  • 描述：统计文件或者目录使用硬盘空间的大小
  • 命令格式 ： du [选项] 文件或目录
常用选项

  • -s 显示文件或目录的大小

  • -h 加一个易读的单位

  • -a 查看所有目录以及文件的大小

[root@localhost ~]# du -sh /etc/passwd

十、find命令

  • 描述：搜索文件或目录

  • 命令格式： find [目录] [条件1] [条件2]

  • 常用的条件

    • -type 类型 （f 文本文件、 d 目录、 l快捷方式）
    • -name “文件名称”
    • -iname 按文档名称查找，且不区分大小写
    • -size + | - 文件大小（k、M、G、TB、PB、EB）
    • -user 用户名
    • -mtime 按修改时间查找
    • -empty 查找空白文件或目录
      • -group 按组查找
[root@localhost ~]# find /boot/ -type f		#查找/boot目录下所有的文件
[root@localhost ~]# find /boot/ -empty 		#查找/boot目录下所有的空文件或目录

  • 高级使用：处理查找的内容

  • 命令格式：find [范围] [条件] -exec 处理命令 {} ;

    • -exec 对找到的文件或目录执行额外的命令
    • -a 并且
    • -o 或者
    • 示例
[root@localhost ~]# find /boot/ -type f  -exec cp {} /opt/ \;
  
[root@localhost ~]# find /opt/ -type f  -exec rm -rf  {}  \;
[root@localhost ~]# ls /opt/
