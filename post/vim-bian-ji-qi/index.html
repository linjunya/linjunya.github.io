vim编辑器

2024-04-04

所有的 Unix Like 系统都会内建 vi 文书编辑器，其他的文书编辑器则不一定会存在。

但是目前我们使用比较多的是 vim 编辑器。

vim 具有程序编辑的能力，可以主动的以字体颜色辨别语法的正确性，方便程序设计。

一、什么是vim

Vim是从 vi 发展出来的一个文本编辑器。代码补完、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。

简单的来说， vi 是老式的字处理器，不过功能已经很齐全了，但是还是有可以进步的地方。 vim 则可以说是程序开发者的一项很好用的工具。

连 vim 的官方网站 (http://www.vim.org) 自己也说 vim 是一个程序开发工具而不是文字处理软件。

三、vi/vim 的基本使用

基本上 vi/vim 共分为三种模式，分别是命令模式（Command mode），输入模式（Insert mode）和底线命令模式（Last line mode）。

  • 简单的说，我们可以将这三个模式想成底下的图标来表示：

3.1、命令模式：

用户刚刚启动 vi/vim，便进入了命令模式。

此状态下敲击键盘动作会被Vim识别为命令，而非输入字符。比如我们此时按下i，并不会输入一个字符，i被当作了一个命令。

以下是常用的几个命令：

  • i 切换到输入模式，以输入字符。
  • x 删除当前光标所在处的字符。
  • : 切换到底线命令模式，以在最底一行输入命令。

若想要编辑文本：启动Vim，进入了命令模式，按下i，切换到输入模式。

命令模式只有一些最基本的命令，因此仍要依靠底线命令模式输入更多命令。

3.1.1、命令模式常用快捷键

命令模式操作快捷键 功能描述
x 删除光标当前所在位置的字符
dd 删除光标当前所在的行
ndd (n为数字)删除包括光标所在行的后n行
d$ 或 D 删除光标位置至行尾的内容，同时删除光标所在位置的字符
C 删除光标位置至行尾的内容，并进入插入模式
d^ 删除光标位置至行首的内容，不删除光标所在位置的字符
J 删除换行符号，将两行合并为一行
u 撤销上一步操作，可以多次使用
r(x) 将光标所在位置的字符替换为（x），（x）可以为任意字符
yy 复制当前行
p（小写） 粘贴至光标所在行的下一行
P（大写） 粘贴至光标所在行的上一行
nyy 复制n行
gg 跳转到首行
G 跳转到尾行
ngg 跳转到第n行
^ 或 Home 跳转到行首
$ 或 End 跳转到行尾
n| 跳转到n列，n表示列号
ZZ 保存退出

3.2、输入模式

在命令模式下按下i就进入了输入模式。

在输入模式中，可以使用以下按键：

  • 字符按键以及Shift组合，输入字符
  • ENTER，回车键，换行
  • BACK SPACE，退格键，删除光标前一个字符
  • DEL，删除键，删除光标后一个字符
  • 方向键，在文本中移动光标
  • HOME/END，移动光标到行首/行尾
  • Page Up/Page Down，上/下翻页
  • Insert，切换光标为输入/替换模式，光标将变成竖线/下划线
  • ESC，退出输入模式，切换到命令模式

3.2.1、进入输入模式的按键及功能

按键 功能描述
a 进入输入模式，后续输入内容将插入至当前光标的后面
A 进入插入模式，后续输入的内容将插入至当前段落的段尾
i 进入插入模式，后续输入的内容将插入至当前光标的前面
I 进入插入模式，后续输入的内容将插入至当前段落的行首
o 进入插入模式，并在当前行的后面创建新的空白行
O 进入插入模式并在当前行的前面创建新的空白行

3.3、底线命令模式（又叫做末行模式）

在命令模式下按下:（英文冒号）就进入了底线命令模式。

底线命令模式可以输入单个或多个字符的命令，可用的命令非常多。

在底线命令模式中，基本的命令：

命令写法 功能描述
:q 退出程序
:w 保存文件
:wq 保存并退出
:q! 放弃更改并且退出
:w /目标文件路径 另存为
:r 目标文件 读入其它文件内容
:set nu | nonu 显示和不显示行号
:set ai | noai 启用和关闭自动缩进

按ESC键可随时退出底线命令模式。

四、vim的查找

4.1、自上往下的查找功能

在命令模式下输入 /

例如：

[root@proxy ~]# head -5 /etc/passwd > /opt/passwd
[root@proxy ~]# vim /opt/passwd
  • 如果个文件中有多个root的字符
    可以通过n跳转至下一个匹配的关键词处
    通过N跳转至上一个关键词处

4.2、实现自下往上的查找

在命令模式输入 ？

例如：

[root@proxy ~]# head -5 /etc/passwd > /opt/passwd
[root@proxy ~]# vim /opt/passwd
  • 此时快捷键N表示查看下一匹配的关键词
  • n表示查看上一匹配的关键词

4.3、忽略大小写

在vim中查找时，可能不清楚关键字的大小写，没办法进行精确的查找时。可以通过命令模式输入：

:set ignorecase				#回车，再输入/ 进行查找

此时可以通过大写的BIN来查找对应的内容

五、vim的替换

5.1、行内替换

指令 功能描述
: s /old/new 替换当前行第一个“old”
: s /old/new/g 替换当前行所有的“old”

5.2、区域内替换

指令 功能描述
:n,m s/old/new/g 替换n到m行所有的old
:% s/old/new/g 替换文件内所有的old

六、vim多窗口编辑

当需要同时编辑多个文档时，分隔窗口可以起到很大的作用。

  • 素材准备
[root@proxy opt]# cat test01.txt 
我是测试文档一
[root@proxy opt]# cat test02.txt 
我是测试文档二
[root@proxy opt]# cat test03.txt 
我是测试文档三
[root@proxy opt]# cat test04.txt 
我是测试文档四

6.1、横向分隔窗口

先用vim 打开 test01.txt

[root@proxy opt]# vim test01.txt

末行模式输入 :split /opt/test02.txt 回车

  • 可以通过 Ctrl+WW切换光标所在位置

  • 在末行模式输入 :close 回车 可以关闭光标所在的文档

6.2、纵向分隔窗口

先用vim 打开 test01.txt

末行模式输入 :vsplit /opt/test02.txt 回车

  • 可以通过 Ctrl+WW切换光标所在位置
  • 在末行模式输入 :close 回车 可以关闭光标所在的文档

6.3、开四个窗口玩一下

  • 切换窗口的快捷键

依然可以通过 Ctrl+WW切换光标所在位置，但切换顺序是按照上图中1、2、3、4的顺序依次切换

Ctrl+Wh跳转到左边的窗口

Ctrl+Wl跳转到右边的窗口

Ctrl+Wj跳转到下面的窗口

Ctrl+Wk跳转到上面一个窗口
