在Linux中安装jdk

2024-04-04
  1. 将下载好的压缩包通过XTerminal复制到root目录下
  2. ls 查看是否复制成功
  3. tar zxvf jdk-8u192-linux-x64.tar.gz
  4. ls 查看解压
  5. mv jdk1.8.0_192/ /usr/local/java 创建一个java文件夹来存放
  6. vi /etc/profile 回车，开始配置环境，按i开始编辑
  7. 移动到最后一行开始添加
  8. export JAVA_HOME=/usr/local/java
  9. export CLASSPATH=:CLASSPATH::CLASSPATH::CLASSPATH:JAVA_HOME/lib/
  10. export PATH=PATH:PATH:PATH:JAVA_HOME/bin
  11. 然后esc，:wq,回车
  12. source /etc/profile 重载资源
  13. java -version 查看java版本

NICE！
